# Luchadores Web

This is a monorepo containing the Luchadores web application.

## Project Structure

- `frontend/` - Next.js frontend application
- `backend/` - Backend application

## Frontend Development

The frontend is a Next.js application. To develop:

```bash
cd frontend
npm install
npm run dev
```

## Deployment

See [VERCEL_DEPLOYMENT.md](./VERCEL_DEPLOYMENT.md) for instructions on deploying the frontend to Vercel.
